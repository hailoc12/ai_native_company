# Dong gop vao AI Native Company — Community Edition

Cam on ban quan tam den viec dong gop! Du day la huong dan de tham gia.

---

## Cach dong gop

### 1. Bao loi (Bug Reports)

Mo [GitHub Issue](../../issues/new) voi thong tin:

- **Mo ta loi**: Loi gi xay ra
- **Buoc tai hien**: Lam the nao de gap loi
- **Moi truong**: OS, Claude Code version, model dang dung
- **Ket qua ky vong**: Ban ky vong dieu gi
- **Ket qua thuc te**: Dieu gi thuc te xay ra

### 2. De xuat cai thien (Feature Requests)

Mo Issue voi nhan `enhancement`:

- **Van de**: Van de gi can giai quyet
- **Giai phap de xuat**: Ban nghi gi se la cach tot
- **Truong hop su dung**: Khi nao tinh nang nay can thiet

### 3. Dong gop Code / Skills

```bash
# 1. Fork repo
# 2. Tao branch
git checkout -b feature/ten-tinh-nang

# 3. Thay doi
#    - Sua / them skill trong skills/
#    - Them vi du trong examples/
#    - Cap nhat tai lieu trong docs/

# 4. Test — chac chan rang:
#    - Skill co SKILL.md hop le
#    - Skill co description dung chuan 4 thanh phan
#    - Folder structure theo KWSR

# 5. Commit
git commit -m "feat: mo ta ngan gon"

# 6. Push + Pull Request
git push origin feature/ten-tinh-nang
```

## Tieu chuan Skill

Moi skill gop phai:

- [ ] Co file `SKILL.md` voi frontmatter dung chuan
- [ ] Co `name`, `description`, `type` trong frontmatter
- [ ] Description co du 4 thanh phan: WHAT / TRIGGER / EXCLUSION / PUSH
- [ ] Co section "When to Use" ro rang
- [ ] Co anti-patterns (nhung gi KHONG lam)
- [ ] Khong chua thong tin nhan cam hoac du lieu nhay cam

---

## Tieu chuan SOP

Moi SOP gop phai:

- [ ] Co 7 section chuan (Tong quan → Phu luc)
- [ ] Co AI integration tags: [AI ASSIST], [AI AUGMENT], [AI WORKFORCE]
- [ ] Co ICOM (Input-Control-Output-Mechanism) day du
- [ ] Co quality gate va escalation path
- [ ] Co cross-link den SOP lien quan
