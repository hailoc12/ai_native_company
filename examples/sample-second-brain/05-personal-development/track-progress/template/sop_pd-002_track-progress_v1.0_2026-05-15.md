## 2. Personal Growth Score (PGS)

PGS = Composite score đo lường growth velocity:

| Dimension | Weight | How Measured | Score 1-10 |
|-----------|--------|-------------|-----------|
| Knowledge Velocity | 30% | Gems extracted / month | <3=1, 3-5=5, >10=10 |
| Application Rate | 30% | Projects using knowledge / month | 0=1, 1=5, ≥3=10 |
| Skill Progress | 25% | Learning goal milestone % | Linear |
| Consistency | 15% | Days with ≥1 capture | <50%=1, 80%=7, 100%=10 |

**PGS = (K×0.3 + A×0.3 + S×0.25 + C×0.15) × 10**
