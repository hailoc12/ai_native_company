## Personal Growth Score (PGS)

PGS = composite score (1-10) updated monthly:
- Knowledge Velocity (30%): gems extracted per month
- Application Rate (30%): projects using knowledge
- Skill Progress (25%): learning goal milestone %
- Consistency (15%): days with ≥1 capture
