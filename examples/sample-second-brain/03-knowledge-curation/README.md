# 03 — Knowledge Curation Department

> **"A vault without structure is a graveyard. Curation is what keeps knowledge alive and findable."**

**Layer:** Operations (L2) — Primary Activity #3
**Head:** Head of Curation
**Team size:** 2

---

## Department IPO

| Component | Detail |
|-----------|--------|
| **INPUT** | Processed notes, gems, syntheses từ Knowledge Processing team |
| **CONTROL** | PARA taxonomy, tag taxonomy, naming convention, link density targets (≥3/note) |
| **OUTPUT** | Well-organized, richly tagged, densely linked vault — searchable in <30 seconds |
| **MECHANISM** | DEVONthink, Obsidian, vault health dashboard, tag taxonomy guide |
