## 1. File Naming Convention

```
Format: [file_type]_[file-name]_v[version]_[YYYY-MM-DD].md

file_type: sop | kpi | charter | policy | template | guide | matrix | flow | report | register

Examples:
  sop_ka-001_capture-source_v1.0_2026-05-15.md
  kpi_ka-001_acquisition-kpis_v1.0_2026-05-15.md
  charter_ka-department_v1.0_2026-05-15.md
```

## 2. Version Control

| Version | When |
|---------|------|
| v1.0 | Initial creation |
| v1.1 | Minor update (add/edit section) |
| v2.0 | Major revision (restructure, change process) |

## 3. SOP Folder Convention

Template READMEs must state:
"⚠️ DO NOT EDIT FILES IN THIS FOLDER DIRECTLY. Copy to input/ first."

## 4. Archive Policy

- Documents older than 1 year + no longer active → move to `_archive/[YYYY]/`
- Never delete — always archive
