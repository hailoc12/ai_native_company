## Build Process Detail

### Step 1: Role Design (2-4 hours)
- Define role profile (responsibilities, goals, KPIs)
- Define personality and communication style
- Map to SOPs and tools
- Define escalation rules

### Step 2: Prompt Engineering (4-8 hours)
- Write system prompt (1,000-3,000 words)
- Include: role definition, company context, task instructions, quality standards, examples
- Test with 10 sample inputs

### Step 3: Knowledge Base Setup (2-4 hours)
- Populate Claude Project with relevant SOPs, templates, brand guides
- Link to product catalog, FAQ, decision trees
- Test retrieval accuracy

### Step 4: Testing & Iteration (4-8 hours)
- Run 20 test inputs covering normal + edge cases
- Score outputs on quality scale (1-5)
- Iterate prompt until average score >= 4.0
- Document known limitations

### Step 5: Deployment (2-4 hours)
- Activate in production
- Set up monitoring (quality checks, escalation)
- First week: CEO reviews every output
- Week 2-4: spot-check 50%
- Month 2+: weekly spot-check
