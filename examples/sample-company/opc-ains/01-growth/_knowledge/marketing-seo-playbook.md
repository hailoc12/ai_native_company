## On-Page SEO Standards

### Blog Post SEO Checklist

| Element | Requirement | Priority |
|---------|------------|----------|
| Title tag | 50-60 chars, primary keyword, compelling | P0 |
| Meta description | 140-155 chars, primary keyword, CTA | P0 |
| URL slug | Short, hyphenated, contains keyword | P0 |
| H1 | 1 per page, contains primary keyword | P0 |
| H2/H3 hierarchy | Logical structure, secondary keywords | P1 |
| First 100 words | Primary keyword naturally included | P1 |
| Internal links | >= 2 per post (to related content) | P1 |
| External links | >= 1 authoritative source | P2 |
| Image alt text | Descriptive, keyword when natural | P1 |
| Word count | >= 800 words (pillar: 2000+) | P1 |
| Readability | Flesch score >= 60 | P2 |
| Schema markup | Article, FAQ, HowTo when applicable | P2 |

### Meta Title Formulas

```
[Number] + [Keyword] + [Benefit]
  → "7 Cach AI Giup SME Tang 40% Hieu Qua"

[Keyword] + [Year] + [Specific]
  → "AI Consulting Vietnam 2026: Bang Gia va Lua Chon"

[How to] + [Keyword] + [Outcome]
  → "Cach Bat Dau Chuyen Đoi So voi AI trong 30 Ngay"

[Keyword] + [Comparison] + [Winner]
  → "ChatGPT vs Claude cho Business: Ai Phu Hop Hon?"
```

## Technical SEO Requirements

### Core Web Vitals Targets

| Metric | Good | Needs Work | Poor |
|--------|------|-----------|------|
| LCP (Largest Contentful Paint) | <2.5s | 2.5-4s | >4s |
| INP (Interaction to Next Paint) | <200ms | 200-500ms | >500ms |
| CLS (Cumulative Layout Shift) | <0.1 | 0.1-0.25 | >0.25 |

### Technical Audit Checklist (Weekly)

- [ ] Crawl errors: 0 critical, <5 warnings
- [ ] Page speed: >=90 (PageSpeed Insights)
- [ ] Mobile-friendly: Pass Google Mobile Test
- [ ] HTTPS: All pages secure
- [ ] XML sitemap: Updated, submitted to GSC
- [ ] Robots.txt: Correctly configured
- [ ] Broken links: <3 per crawl
- [ ] Duplicate content: No issues
- [ ] Canonical tags: Properly set
- [ ] Schema markup: Validated

### Schema Markup Types (by content)

| Content Type | Schema | Fields |
|-------------|--------|--------|
| Blog post | Article | title, author, datePublished, image |
| FAQ | FAQPage | question, acceptedAnswer |
| How-to guide | HowTo | step, name, text |
| Service page | Service | name, description, provider |
| Testimonial | Review | author, reviewRating, reviewBody |

## Content Structure Templates (SEO-Optimized)

### Pillar Page Structure (2000+ words)

```
H1: [Primary Keyword] — Complete Guide [Year]

Introduction (200 words)
- Hook + pain point
- What reader will learn
- Why this matters NOW

H2: [Keyword Variation 1]
- Definition/explanation
- Data/statistics
- Examples

H2: [Keyword Variation 2]
- Step-by-step process
- Visual aids
- Tips & warnings

H2: [Keyword Variation 3]
- Tools/resources
- Comparison table
- Recommendations

H2: Common Questions (FAQ schema)
H3: Question 1 → Answer
H3: Question 2 → Answer
H3: Question 3 → Answer

H2: Conclusion
- Key takeaways
- Next steps
- CTA

Meta: Internal links >= 3, External links >= 2
```

### Standard Blog Post Structure (800-1200 words)

```
H1: [Keyword-optimized title]

Hook paragraph (2-3 sentences, include keyword)

H2: [Problem/Context]
H2: [Solution/Method]
H2: [Implementation/Examples]
H2: [Key Takeaways]

CTA paragraph

Meta: Internal links >= 2, External links >= 1
```

## Link Building Strategy

### Internal Linking Rules

1. Moi blog post phai link toi >= 2 bai viet khac tren ains.vn
2. Link voi anchor text tu nhien (khong exact-match spam)
3. Link tu pillar pages toi cluster content
4. Link tu new content toi high-performing old content

### External Link Building Tactics

| Tactic | Priority | Effort | Expected Result |
|--------|----------|--------|----------------|
| Guest posting on VN tech blogs | P1 | High | 2-3 backlinks/thang |
| HARO/Quoted expert | P2 | Medium | 1 backlink/thang |
| Directory listings (VN business) | P1 | Low | 3-5 listings total |
| Partnership links | P2 | Medium | 1-2 backlinks/thang |
| Social sharing (indirect) | P3 | Low | Brand signals |
