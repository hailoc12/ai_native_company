# AINS Reports — Index

## Tong quan
Thu vien bao cao dinh ky cua AINS. Moi bao cao duoc tao boi AI Workers va review boi CEO.

## Cau truc

| Thu muc | Tan suat | Noi dung |
|---------|----------|----------|
| `weekly/` | Hang tuan | CEO brief, operations summary |
| `monthly/` | Hang thang | Business review, financial summary, KPI scorecard |
| `quarterly/` | Hang quy | Strategic review |

## Quy tac
- File naming: `[type]_[period].md` (vi du: `weekly-ceo-brief_2026-W18.md`)
- Moi bao cao co metadata header
- CEO review va sign-off truoc khi file
- Bao cao cu duoc giu lai trong Git history
