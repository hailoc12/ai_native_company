# Case Study: TechCorp Vietnam — AI Readiness Assessment & Strategy

**Client**: TechCorp Vietnam
**Industry**: Software Outsourcing
**Project Duration**: 3 tuần (March 10-31, 2026)
**Contract Value**: $5,500
**Status**: Completed

---

## Bối cảnh

TechCorp Vietnam là công ty software outsourcing 45 nhân sự tại Q1, TP.HCM. Revenue ~$2.5M/year. CEO Nguyễn Minh Tuấn muốn áp dụng AI để improve internal processes nhưng không biết bắt đầu từ đâu.

### Vấn đề của khách hàng
- Đã thử ChatGPT team plan nhưng team không biết apply vào workflow
- Tham gia 2 AI workshop nhưng "toàn lý thuyết"
- Tech team hào hứng nhưng management cần thấy ROI rõ ràng
- Đã tiếp cận FPT AI nhưng quote $15,000+ và timeline 8 tuần

---

## Giải pháp của AINS

### Phase 1: AI Readiness Assessment (Tuần 1)
- **Activity**: Assess current AI maturity, data readiness, team capability
- **Deliverable**: AI Readiness Report với scoring matrix
- **Method**: Interview 5 key stakeholders + audit tech stack + review data infrastructure

**Findings**:
| Dimension | Score (1-5) | Ghi chú |
|-----------|-------------|---------|
| Data Readiness | 3.5 | Có data nhưng fragmented |
| Tech Infrastructure | 4.0 | Git, CI/CD tốt, thiếu ML ops |
| Team AI Skill | 2.0 | Biết dùng ChatGPT basic, không deeper |
| Leadership Buy-in | 3.0 | CEO muốn, board cần thấy ROI |
| Process Maturity | 3.5 | Agile tốt, thiếu AI-specific process |

### Phase 2: AI Strategy & Roadmap (Tuần 2)
- **Activity**: Define AI use cases, prioritize, build 6-month roadmap
- **Deliverable**: AI Strategy Document + Implementation Roadmap
- **Method**: Workshop với tech leads + prioritize theo impact/effort matrix

**Top 3 Use Cases Identified**:
1. **AI Code Review** — Estimate: tiết kiệm 30% review time ($2,400/tháng)
2. **AI Testing Automation** — Estimate: giảm 40% manual testing ($1,800/tháng)
3. **AI Project Estimation** — Estimate: improve accuracy 25%, giảm overruns

### Phase 3: Pilot Implementation — Code Review AI (Tuần 3)
- **Activity**: Implement AI-powered code review cho 1 project pilot
- **Deliverable**: Working prototype + team training
- **Tools**: Cursor + Custom rules + CI/CD integration

**Pilot Results**:
| Metric | Before | After (Pilot) | Improvement |
|--------|--------|---------------|-------------|
| Code review time | 4.2h/sprint | 2.8h/sprint | -33% |
| Bug detection rate | 72% | 85% | +13pt |
| Developer satisfaction | 3.2/5 | 4.1/5 | +28% |

## Lessons Learned

### What went well
1. Speed — proposal 2h, delivery 3 tuần (vs. industry 6-8 tuần)
2. AI Workers cho phép deliver multiple workstreams song song
3. Pilot approach chứng minh ROI nhanh, build trust

### What could improve
1. Nên thêm training session cho management (không chỉ tech team)
2. Document handover cần chi tiết hơn (TechCorp tech lead cần support sau handover)
3. Pricing có thể cao hơn — client暗示 sẽ trả $7,000-8,000

### SOPs Updated
- Updated `SOP-DEL-001` (Project Onboarding) — thêm management training step
- Updated `SOP-DEL-005` (Client Handover) — thêm detailed documentation requirement
- Created `SOP-DEL-008` (Pilot Implementation) — new SOP cho pilot projects
