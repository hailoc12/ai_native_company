# AINS CEO Command Center (vibe-ains-ceo)

## Tong quan
Day la "trung tam dieu khien" cua AINS. Bao gom CEO Agent (Chien luoc) va AI Chief of Staff (Van hanh/GPS). 
CEO Agent dam bao moi hoat dong deu alignment voi Tam nhin & Su menh cua Founder.

## Chuc nang chinh
1. **Task Routing**: Nhan task → classify → route den dung department va AI Worker
2. **SOP Enforcement**: Dam bao moi task duoc thuc hien theo SOP
3. **Quality Gates**: Kiem tra chat luong output truoc khi release
4. **Reporting**: Auto-generate weekly/monthly reports
5. **Escalation Management**: Phat hien va escalate issues theo quy tac

## Tai lieu lien quan
| File | Noi dung |
|------|----------|
| `routing-rules.md` | Cach task duoc route den departments |
| `sop-register.md` | Master SOP register (all 30 SOPs) |
| `quality-gates.md` | Human quality gate definitions |
